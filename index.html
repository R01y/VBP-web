Virtual Body Profile

What is virtual body profile?

It is a website that can provide a converter to convert your body data to
a cloth size that matched your body
we also provide a table that allows you to check it by yourself



How to use it?

We provide a table of international cloth size
you just need to type in the data that we need
then click submit, the size will pop out on the bottom of the page

Do I need to type in those data again and again?

Fortunately, you just need to submit those data once!
we have a refill button that allows the website
to find and refill the data, which you submit last time
